System Architecture

ScraperSky is a FastAPI application running in a Docker container that provides web scraping, data extraction, and analysis capabilities.

Core Technologies:

  • Backend: FastAPI (Python web framework)
  • Database: PostgreSQL via Supabase
  • Authentication: JWT with Supabase integration
  • Containerization: Docker & Docker Compose
  • Deployment: Kubernetes or Render

Service Architecture:

  • Auth Service: JWT token validation, user context
  • DB Service: Database connection management
  • Error Service: Centralized error handling
  • Job Manager: Background task orchestration
  • Tenant Isolation: Multi-tenant data separation

API Overview

The ScraperSky API provides endpoints for web scraping, metadata extraction, and data analysis.

Core Services:

  • Sitemap Analyzer: Extract and analyze website sitemaps
  • Email Scanner: Extract email addresses from websites
  • Places Scraper: Extract location data
  • Contact Manager: Organize and manage extracted contacts
  • Sitemap Analyzer: Visualize and analyze website structure
